## 下一步操作

### 1. 安装 Go 环境
```bash
# 下载并安装 Go 1.21+
# https://golang.org/dl/
```

### 2. 配置数据库
```sql
-- 创建 PostgreSQL 数据库
CREATE DATABASE map_tile_system;
```

### 3. 启动后端
```bash
cd backend
go mod download
go run cmd/server/main.go
```

### 4. 启动前端（已在运行）
前端开发服务器已经在运行：http://localhost:5174

### 5. 前后端联调
- 前端会自动请求 `/api` 接口
- 后端运行在 `http://localhost:8080`
- 需要配置前端代理或修改 API 地址
